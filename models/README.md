# Trained Models

Trained models are not included in this repository due to file size constraints (SB3 `.zip` files exceed GitHub's limits).

## Model availability
Models are available upon request — contact via [LinkedIn](https://linkedin.com/in/iker-alaminos-hernandez-iarobots) or [iarobots.tech](https://iarobots.tech).

## Phase reference table
See the main README for the commit hash corresponding to each training phase (1–4 agents + Lite6).

## To reproduce training
Run the scripts in `src/multiagente/` for the satellite formation, or `src/lite6/` for the Lite6 manipulator. Models will be saved automatically to this directory.
